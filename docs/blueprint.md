# **App Name**: HopeEngineer Hub

## Core Features:

- Animated Hero Section: Implement live animations in the hero section to create a lively and engaging first impression.
- Blog Page: Create a blog page with the ability to create, edit, delete, and import articles from Medium and Substack. Includes role-based access for article management (Admin only).
- Work With Me Page: Showcase the 'hopeengineer' products and services: 1-on-1 sessions, Full Circle (FC) Skool community, and AI content automation services.
- Client Testimonials: Display client testimonials as screenshots to maintain authenticity and visual appeal.
- AI Apps Page: Dedicate a page to share the AI apps developed under the 'hopeengineer' brand.
- About Me Page: Develop an 'About Me' page including contact details and links to social media accounts (Threads, Instagram, Facebook).

## Style Guidelines:

- Dark theme with a primary color of Blood Orange (#8B0000), reflecting the brand's core identity. The dark theme contrasts with Blood Orange to communicate both sophistication and warmth.
- Background color: Very dark grayish-red (#1A0000), a desaturated version of the primary color.
- Accent color: Moderate orange (#CD6600), which will contrast clearly and draw attention to CTAs without clashing.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text, ensuring readability and a modern feel.
- Implement scroll-based animations and cursor responsiveness to enhance user engagement and create a dynamic browsing experience.
- Design a responsive layout ensuring seamless user experience across all devices.
- Use simple and clear icons to aid navigation and comprehension. If any code is displayed, use 'Source Code Pro' (monospaced) font.
