# HopeEngineer Hub

A personal website built with Next.js 15, Supabase (Auth + PostgreSQL), and shadcn/ui.

## Getting Started

```bash
npm install
npm run dev
```

## Stack

- **Framework**: Next.js 15 (Turbopack)
- **Database & Auth**: Supabase
- **Styling**: Tailwind CSS + shadcn/ui
- **Hosting**: Railway
